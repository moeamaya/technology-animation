Aptible Container Platform

The Aptible Container Platform is a deployment platform designed to securely deploy Docker containers into HIPAA-ready environments. We manage zero-downtime deployments, scale your clusters, encrypt and back up your data, and run your ops, allowing you to focus on development.

Secure Networks

Each PHI-ready stack runs in its own AWS Virtual Private Cloud. Databases and internal services run in a private subnet, inaccessible from the Internet. All stack instances are configured with deny-by-default firewalls.

Easy, Secure Load Balancing

Each SSL/TLS endpoint receives a dedicated load balancer, scaling traffic across your app containers. SSL/TLS endpoints make it easy to manage x509 SSL/TLS certificates, force SSL/TLS connections, and create private services. Each endpoint receives automatic security updates on a regular basis.

Designed for Portability

Achieve dev/prod parity by standardizing your apps across environments. Support for Docker and the most popular open source languages, frameworks, and databases means you never have to worry about lock-in.

Easy Developer Access

Robust role-based access controls make it easy to authorize and audit your team's activity. Use the CLI tool or web dashboard to manage your stack, including SSH and tunnel access. Script and automate common actions with our ops API.

Secure Databases

Databases run in a private subnet, hidden from the outside Internet. Access is restriced to your apps and team members. Database traffic is encrypted in transit, and data is encrypted at rest, using FIPS 140-2 approved modules and Aptible-managed keys. Nightly backups are automatically taken and stored in a separate geographic region.

Perform at Scale

Easily scale apps, domains, and databases with the security and reliability of the AWS cloud. Because your PHI-ready stacks are dedicated to you, the Aptible Container Service has no performance tradeoff. Zero-downtime deploys make scaling seamless.
